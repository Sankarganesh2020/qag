
### How to setup?

### Create python Environment
```bash
python -m venv .venv
.venv\Scripts\activate
```

### install requirements
```bash
pip install -r requirements.txt
```

### Create .env file and update the below Environment variables
```bash
AZURE_OPENAI_ENDPOINT = ""
AZURE_OPENAI_API_KEY  = ""
```

### Run the app
```bash
python app.py
```
